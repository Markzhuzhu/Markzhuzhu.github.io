# 一、题目

> 这道超级简单的题目没有任何输入。
> 
> 你只需要把这句很重要的话 —— “I'm gonna WIN!”——连续输出三遍就可以了。

注意每遍占一行，除了每行的回车不能有任何多余字符。

**输入格式：**
> 无

**输出格式：**



> I'm gonna WIN! 
> I'm gonna WIN!
>  I'm gonna WIN!

 

# 二、解决方案
# 1.思路
这道题很简单，我们直接打印，注意换行即可。

——————————————
# 2.代码

```c
#define _CRT_SECURE_NO_WARNINGS 1
#include <stdio.h>
int main()
{
	for (int i = 0; i < 3; i++)
	{
		printf("I'm gonna WIN!\n");
	}
	return 0;
}
```
