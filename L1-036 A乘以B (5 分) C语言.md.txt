# 一、题目

> 看我没骗你吧 —— 这是一道你可以在 10 秒内完成的题：给定两个绝对值不超过 100 的整数 A 和 B，输出 A 乘以 B 的值。
> 
> 
**输入格式：**
> 输入在第一行给出两个整数 A 和 B（−100≤A,B≤100），数字间以空格分隔。


**输出格式：**

> 在一行中输出 A 乘以 B 的值。

**输入样例1：**
-8 13

**输出样例1：**

-104




 

# 二、解决方案
# 1.思路
①这道题我们直接按题目要求写出来即可。

——————————————
# 2.代码

```c
#define _CRT_SECURE_NO_WARNINGS 1
#include <stdio.h>
int main()
{
	int a, b;
	scanf("%d %d", &a, &b);
	printf("%d", a * b);
	return 0;
}
```
