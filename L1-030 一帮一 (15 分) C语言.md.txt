# 一、题目

>“一帮一学习小组”是中小学中常见的学习组织方式，老师把学习成绩靠前的学生跟学习成绩靠后的学生排在一组。本题就请你编写程序帮助老师自动完成这个分配工作，即在得到全班学生的排名后，在当前尚未分组的学生中，将名次最靠前的学生与名次最靠后的异性学生分为一组。
> 
> 
**输入格式：**
>输入第一行给出正偶数N（≤50），即全班学生的人数。此后N行，按照名次从高到低的顺序给出每个学生的性别（0代表女生，1代表男生）和姓名（不超过8个英文字母的非空字符串），其间以1个空格分隔。这里保证本班男女比例是1:1，并且没有并列名次。


**输出格式：**

> 每行输出一组两个学生的姓名，其间以1个空格分隔。名次高的学生在前，名次低的学生在后。小组的输出顺序按照前面学生的名次从高到低排列。

**输入样例1：**
8
0 Amy
1 Tom
1 Bill
0 Cindy
0 Maya
1 John
1 Jack
0 Linda

**输出样例1：**

Amy Jack
Tom Linda
Bill Maya
Cindy John




 

# 二、解决方案
# 1.思路
①我们用二维数组存放每一个学生的姓名，同时再定义一个int数组来存放每个学生的性别，然后按照性别依次匹配即可，题目帮我们把成绩已经排序了，所以简单了不少
②只用注意不用重复就可，把匹配过的人除去即可

——————————————
# 2.代码

```c
#define _CRT_SECURE_NO_WARNINGS 1
#include <stdio.h>
int main()
{
	int n = 0;
	scanf("%d", &n);
	int arr[50] = { 0 };
	char arr1[50][20];
	for (int i = 0; i < n; i++)
	{
		scanf("%d %s", &arr[i], arr1[i]);
	}
	int j = 0;
	for (int i = 0; i < n ; i++)
	{
		for ( j = n - 1; j >= 0; j--)
		{
			if ( arr[i]+arr[j]==1)
			{
				printf("%s %s\n", arr1[i], arr1[j]);
				arr[j] = 2;
				break;
			}
		}
	}
	return 0;
}
```
