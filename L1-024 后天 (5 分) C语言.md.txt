# 一、题目

> 如果今天是星期三，后天就是星期五；如果今天是星期六，后天就是星期一。我们用数字1到7对应星期一到星期日。给定某一天，请你输出那天的“后天”是星期几。
> 
> 
**输入格式：**
> 输入第一行给出一个正整数D（1 ≤ D ≤ 7），代表星期里的某一天。


**输出格式：**

> 在一行中输出D天的后天是星期几。


**输入样例1：**

3

**输出样例1：**

5



 

# 二、解决方案
# 1.思路
这道题比较简单，我的做法是把周一--周日分别想成数组的下标，然后1对应3,2对应4，只要注意把数组的首元素的值设为0即可
——————————————
# 2.代码

```c
#define _CRT_SECURE_NO_WARNINGS 1
#include <stdio.h>
int main()
{
	int arr[8] = { 0,3,4,5,6,7,1,2 };
	int n = 0;
	scanf("%d", &n);
	printf("%d", arr[n]);
	return 0;
}
```
