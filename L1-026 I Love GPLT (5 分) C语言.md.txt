# 一、题目

> 这道超级简单的题目没有任何输入。

你只需要把这句很重要的话 —— “I Love GPLT”——竖着输出就可以了。

所谓“竖着输出”，是指每个字符占一行（包括空格），即每行只能有1个字符和回车。

**输入格式：**
> 无
> 



**输出样例2：**

I
 
L
o
v
e
 
G
P
L
T

 

# 二、解决方案
# 1.思路
我们把题目的那句话放在一个字符数组里直接用for循环打印，注意换行即可。

——————————————
# 2.代码

```c
#define _CRT_SECURE_NO_WARNINGS 1
#include<stdio.h>
int main()
{
    char a[12] = { "I Love GPLT" };
    int i;
    for (i = 0; i <= 11; i++)
        printf("%c\n", a[i]);
    return 0;
}
```
