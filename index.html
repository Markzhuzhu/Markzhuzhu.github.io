一、创建一个Github账号
1.进入github网页，右上角有个sign in有账号可以直接登入，没有他会提示我们注册一个账号

2，接下来创建一个库


3.进入github pages


4.弄到这里我们建库就基本完成了，至于换主题就先不换，没什么意义而且也容易出错，我第一次就在换主题上栽跟头，弄了好久还没弄好，所以建议就不要换主题了。
二、下载安装Git 和 node.js
可以去官网 找下载安装包 ，下载好之后
我们可以win+R打开 输入 cmd，之后分别输入node -v npm -v ，确保node.js环境配置好
下载好git后鼠标右键 会出现
三、安装HEXO
1.在合适的地方创建一个文件夹，右键 选择Git bash here，
2.输入 npm install -g hexo进行安装
3.输入hexo init 这个过程可能稍稍有点长 耐心等一下

出现了start blogging with Hexo 这一步 就好了
4.输入 npm install,安装所需组件
5.输入 hexo g
6.hexo s

7.进入这个网页

三、部署到github
1、编辑根目录下_config.yml文件,添加如下代码(Markzhuzhu换成自己的名字进行保存，这个文件选择用记事本打开，进去之后就可以修改了）
deploy:
type: git
repo: git@github.com:Markzhuzhu/Markzhuzhu.github.io
branch: master
2.安装一个扩展
npm install hexo-deployer-git –save

3.输入cd ~/.ssh，出现这个说明不存在.ssh

4.输入$ ssh-keygen -t rsa -C “xxxxxxx@qq.com“(自己邮箱)

之后再进行上图中操作
四、添加SSH Key到Github
1、点击“settings”

2、点击“SSH and GPG keys”,然后点击“New SSH key”新建SSH keys

3、填写key
注意这个文件的的key 是你用记事本打开id_ras.pub里的东西，直接复制粘贴过来即可

4、输入ssh -T git@github.com，测试添加ssh是否成功。如果看到Hi后面是你的用户名，就说明成功了（有一个会让你输入 yes）

5、设置你的账号信息
最后设置你的账号信息即可，比如下图

6.部署到你的github上
输入 hexo d -g 即可

五、用csdn，使用markdown上传你的hexo上
导出这个文件，为md格式,找到你一开始创建文件的地方，点source 然后点_posts ，将这个你要用csdn上传的文件放入，比如我要上传这个猜数字游戏，这样加进去之后
然后在你原本的文件下 再Git bash Here
输入hexo clean
hexo g
hexo d
hexo s
在进入http://localhost:4000，这样就可以看到我们新在hexo上传上传的东西了

#include <stdio.h>
#include <time.h>
#include <stdlib.h>
void game()
{
int guess=0;
printf(“请输入一个数\n”);
int r=rand()%100+1;//生成一个1-100的随机数
while(1)
{
scanf(“%d”,&guess);
if(guess<r)
{
printf(“猜小了\n”);

    }
    else if(guess>r)
    {
      printf("猜大了\n");
    
    }
    else 
    {
    printf("恭喜你，猜对了\n");
    break;
    }    
}

}
menu()
{
printf(“******************\n”);
printf(“ 1.play \n”);
printf(“ 0.exit *****\n”);
printf(“***********************\n”);
}
int main()
{
int input=0;
srand((unsigned int)time(NULL));
do
{
menu();
printf(“请选择：> “);
scanf(“%d”,&input);
switch(input)
{
case(1):
printf(“玩游戏\n”);
game();
input = 0;
break;
case(0):
printf(“退出游戏\n”);
break;
default:
printf(“请输入1或2\n”);
break;
}
}
while(input);
return 0;
}
